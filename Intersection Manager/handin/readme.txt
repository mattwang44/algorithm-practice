8. 執行、使用方式說明：
   主程式：
    編譯完成後，在檔案目錄下會產生一個 build 的執行檔
    執行檔的命令格式為：

        ./built <input file name> <output file name>

    例如：

        ./build test.in test.out

    便會在目錄下自動生成輸出檔test.out
